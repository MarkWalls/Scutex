Scutex  - Alpha/Beta
Scutex is a free Microsoft .Net Framework 4.0 based licensing and trial system for use in your GUI (Winform/WPF) based desktop applications and .Net Framework components and controls. Scutex is currently in pre-release Alpha/Beta mode and should only be used for evaluation and testing purposes. Please check the Scutex website for frequent updates and report any issues or bugs that you may encounter so they can be fixed as soon as possible.
Scutex requires the Microsoft .Net Framework 4.0.
